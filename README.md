# VoidRifts 🌌
> Where Content Breaks Through.

VoidRifts is a high-fidelity, cyberpunk-inspired video streaming platform. Built as a next-generation "Youtube Lite", it focuses on immersive aesthetics, neural-linked interactions, and a seamless viewing experience within the drift.

## 🚀 Vision
The void is not empty; it's waiting to be filled. VoidRifts provides the interface to broadcast signals across the net, allowing creators to bridge the gap between reality and digital expression.

## 🛠 Tech Stack
- **Frontend**: Next.js 15+, Tailwind CSS 4, React Icons
- **Backend**: Node.js, Express, MongoDB, Cloudinary
- **Auth**: JWT, BcryptJS
- **Real-time**: Socket.io (Experimental)

## 📡 Key Features
- **Signal Transmissions**: High-end video cards with ambient glow effects.
- **Neural Linking**: User profiles with verified agent stats.
- **Frequency Search**: Advanced identification of data signals.
- **Breach Reporting**: Complete system for Identifying and reporting transmission anomalies.
- **Sync Feed**: Interactive comment sections rebranded for the void.

## ⚙️ Development Setup

### Backend
1. `cd backend`
2. `npm install`
3. Configure `.env` with your MongoDB and Cloudinary credentials.
4. `npm run dev`

### Frontend
1. `cd frontend`
2. `npm install`
3. Configure `.env.local` with `NEXT_PUBLIC_API_URL`.
4. `npm run dev`
